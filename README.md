# 게임 정보 카드 생성기 (Game Info Card Generator)

기사 본문에 삽입할 게임 정보 카드 HTML을 폼 입력만으로 생성하는 단일 파일 도구입니다.
한국어/영어 양쪽으로 동작하며, 결과물은 HTML 코드와 740px PNG 이미지 두 가지 형태로 받을 수 있습니다.

## 미리보기 출력 예시

생성되는 카드는 다음과 같은 구조를 가집니다.

- 상단 16:9 대표 이미지
- 좌측 그라데이션 컬러 바
- 한글명(영문명) 형식의 타이틀
- 개발사 / 퍼블리셔 / 플랫폼 / 플레이 / 출시일 / 키워드 6개 항목 그리드

## 주요 기능

- **단일 HTML 파일** — 외부 의존성은 `html2canvas` CDN 하나뿐. 더블클릭만으로 브라우저에서 바로 실행.
- **한국어 / 영어 토글** — UI 텍스트와 카드 내부 라벨(`개발사` ↔ `Developer` 등) 동시 전환. 선택 결과는 브라우저에 저장되어 다음 접속 시에도 유지.
- **HTML 코드 복사** — 인라인 스타일 기반의 `<span>` 구조라 CMS·메일에 그대로 붙여넣기 가능.
- **PNG 이미지 다운로드** — 화면 폭과 무관하게 항상 가로 740px로 출력. 파일명은 입력한 게임명으로 자동 지정.
- **CORS 자동 처리** — 외부 이미지가 직접 fetch되지 않으면 자동으로 CORS 프록시 경유.

## 사용 방법

1. `game-card-generator.html` 파일을 더블클릭해서 브라우저로 엽니다.
2. 상단에서 한국어 / English 중 원하는 언어를 선택합니다.
3. 게임 정보를 입력합니다. (대표 이미지 URL, 게임명, 개발사, 퍼블리셔, 플랫폼, 플레이, 출시일, 키워드)
4. **코드 생성** 버튼을 누르면 우측에 미리보기와 HTML 코드가 나타납니다.
5. **코드 복사** 또는 **이미지 다운로드** 버튼으로 결과물을 가져갑니다.

처음 사용해보신다면 **예시 채우기** 버튼을 누르면 샘플 데이터가 자동 입력됩니다.

## 동료에게 공유하기

가장 간단한 방법은 `game-card-generator.html` 파일을 카카오톡·슬랙·메일 등으로 전달하는 것입니다.
별도 설치 없이 받은 사람이 브라우저로 더블클릭만 하면 바로 사용할 수 있습니다.

저장소 URL을 공유해도 됩니다. 받은 사람이 GitHub에서 `Code → Download ZIP`으로 받거나, 파일 페이지에서 우측 상단 다운로드 버튼으로 받을 수 있습니다.
